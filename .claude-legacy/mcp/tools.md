# 도구 맵

## 로컬 도구

- `rg`: 파일 검색
- `git`: branch, diff, commit, worktree 관리
- `gh`: GitHub Issue, PR, review comment 관리
- `python`: CV pipeline, experiment, visualization 실행

## 연구 도구

- 공식 문서
- 논문 PDF
- GitHub repository
- Hugging Face model card

## 향후 MCP 후보

- GitHub
- Notion 또는 wiki
- experiment tracker
- artifact storage

## 원칙

반복적이고 실수하기 쉬운 작업만 도구화한다. 단순 문서 읽기는 파일 검색으로 충분하다.
